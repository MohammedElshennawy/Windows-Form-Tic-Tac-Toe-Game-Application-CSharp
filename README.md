# 🎮 Tic-Tac-Toe Game

A simple desktop Tic-Tac-Toe game built using **C#** and **Windows Forms** with an interactive graphical user interface.

## 🚀 Features

* 2-player gameplay
* Dynamic turn tracking
* Winner detection system
* Game status updates
* Restart game functionality
* Interactive GUI using Windows Forms

## 🧠 Concepts Practiced

* Event-Driven Programming
* Windows Forms Development
* Game Logic & State Management
* Conditional Statements
* Dynamic UI Updates
* Working with Buttons and Labels

## 🛠️ Technologies Used

* C#
* .NET Framework
* Windows Forms

This project was built as part of my learning journey in desktop application development and GUI programming using C# and .NET.
